# Структура миграций базы данных

## Основные таблицы

### 1. flights - Рейсы
Основная таблица для хранения информации о рейсах.
- `id` - Уникальный идентификатор
- `activity_id`, `activity_type`, `activity_subtype` - Тип активности
- `aircraft_type`, `aircraft` - Тип и регистрация ВС
- `flight_number` - Номер рейса
- `departure_airport`, `arrival_airport` - Аэропорты вылета и прилета
- `date_departure`, `time_departure` - Дата и время вылета
- `date_arrival`, `time_arrival` - Дата и время прилета
- `actual_date_departure`, `actual_time_departure` - Фактические дата и время вылета
- `actual_date_arrival`, `actual_time_arrival` - Фактические дата и время прилета
- `Captain`, `FO`, `PilotInstructor` - Члены экипажа
- `AddCrew1`, `AddCrew2` - Дополнительные члены экипажа
- `SeniorFlightAttendant`, `FlightAttendant1`, `FlightAttendantInstructor` - Бортпроводники
- `AviaPersonnel1`, `AviaPersonnel2`, `AviaPersonnel3` - Авиационный персонал
- `trip_number` - Номер рейса
- `passengers_count` - Количество пассажиров
- `checklist_completed` - Чек-лист выполнен
- `status` - Статус рейса
- `note` - Примечания

### 2. crews - Экипаж
Таблица для хранения информации о членах экипажа.
- `id` - Уникальный идентификатор
- `Name` - ФИО
- `ShortName` - Краткое имя
- `Position` - Должность
- `BirthDate` - Дата рождения
- `Address` - Адрес
- `Phone` - Телефон
- `email` - Email
- `Description` - Описание
- `active` - Активность

### 3. fleets - Воздушные суда
Таблица для хранения информации о воздушных судах.
- `id` - Уникальный идентификатор
- `aircraft_number` - Номер ВС
- `Owner` - Владелец
- `Type` - Тип ВС
- `Class` - Класс ВС
- `Pax_number` - Количество пассажиров
- `Airport_base` - Базовый аэропорт
- `Date_manufacture` - Дата производства
- `Repair` - Ремонт
- `Description` - Описание
- `Height`, `Length`, `Wing` - Размеры
- `Cruise_speed`, `Range` - Летные характеристики
- `active` - Активность

### 4. aircrafts_types - Типы воздушных судов
Таблица для хранения типов ВС.
- `id` - Уникальный идентификатор
- `iata`, `icao`, `rus` - Коды типов ВС
- `name_eng`, `name_rus` - Названия на английском и русском
- `group` - Группа
- `crew1`, `crew2` - Экипаж
- `country_manufacture` - Страна производства
- `wingspan`, `long` - Размеры
- `helicopter` - Вертолет
- `active` - Активность

## Индексы

Все основные таблицы имеют индексы для оптимизации запросов:
- Индексы по основным полям поиска
- Индексы по статусам и флагам активности
- Составные индексы для связующих таблиц
- Уникальные индексы для предотвращения дублирования

## Связи

- `flights` ↔ `flight_crews` ↔ `crews` - Связь рейсов с экипажем
- `flights` ↔ `flight_pax` ↔ `passengers` - Связь рейсов с пассажирами
- `flights` ↔ `flight_readiness_type` ↔ `readiness_types` - Связь рейсов с готовностью
- `flights` ↔ `flight_readiness` - Связь рейсов с готовностью
- `flights` ↔ `passengers` - Прямая связь рейсов с пассажирами
